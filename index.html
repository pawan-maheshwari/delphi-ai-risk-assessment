This instrument turns the MIT AI Risk Repository’s evidence base into a working assessment. Each respondent rates 24 risks across seven domains; the engine computes a residual-risk score and the responses accumulate in a shared store — up to 1,000 — that a live dashboard re-aggregates into one organisational view as each new submission arrives.

SURVEY

Each user rates 24 risks

Likelihood, impact severity (1–5 negligible→catastrophic), and control maturity — across all seven MIT domains.

STORE

Responses persist & pool

Every submission is written to a shared internal store, capped at 1,000, so many users’ surveys roll into a single organisational dataset.

LIVE

The dashboard updates itself

A condensed report re-computes on a live cycle, folding each new response into the running risk picture and benchmarking it against the expert panel.
